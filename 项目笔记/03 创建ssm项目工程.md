# 创建ssm项目工程

## 3.1 创建maven工程

* 1、打开IDEA->file->new->project
* 2、选择Maven，勾选Create from archetype，
    选择org.apache.maven.archetypes:maven-archetype-webapp

## 3.2 使用自己安装的maven
* 更改Maven home directory
* 更改User settings file（maven/conf/settings.xml）
* 添加Properties{archetypeCatalog:internal}

## 3.4 完善目录结构
* 在main文件夹下新建java和resources目录
* 分别右击选择Mark Directory as 
* 点击Sources Root和Resources Root

## 3.5 目录进行分层
##### 在main.java文件夹中新建目录如下
* com.gychen.controller
* com.gychen.dao
* com.gychen.po
* com.gychen.service
* com.gychen.util
